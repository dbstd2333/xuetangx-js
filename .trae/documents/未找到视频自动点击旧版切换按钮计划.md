# 计划：未找到视频节点时自动点击"旧版"切换按钮

## 任务目标

当脚本检测不到视频节点时（`video === undefined`），先尝试点击页面上的"旧版"切换按钮，切换到旧版界面后再检测视频节点，而不是直接跳转到下一章节。

## 修改步骤

### 1. 修改"未找到视频"的处理逻辑

**位置**：第384-389行

**原代码**：

```javascript
// --- 当前不是视频：当作作业/讨论，直接跳下一节 ---
if (video === undefined) {
    console.log("未找到视频播放器，可能是作业/讨论，跳转下一个未完成章节。");
    logStatus("当前章节不是视频（可能是作业/讨论），跳到下一个未完成章节。");
    gotoNextUnfinished(index);
    return;
}
```

**修改为**：

```javascript
// --- 当前不是视频：尝试切换到旧版界面 ---
if (video === undefined) {
    var versionSwitch = document.querySelector('.version-switch');
    if (versionSwitch) {
        console.log("未找到视频播放器，尝试点击旧版切换按钮...");
        logStatus("未找到视频播放器，点击旧版切换按钮。");
        versionSwitch.click();
        // 给界面切换一点时间，然后重新检测视频
        setTimeout(function() {
            var videosRetry = document.getElementsByClassName("xt_video_player");
            var videoRetry = videosRetry.length > 0 ? videosRetry[0] : undefined;
            if (videoRetry !== undefined) {
                console.log("切换旧版后成功找到视频播放器。");
                logStatus("切换到旧版后找到视频，继续播放。");
                next(); // 重新执行检测流程
                return;
            }
        }, 1500);
        return;
    }
    console.log("未找到视频播放器，可能是作业/讨论，跳转下一个未完成章节。");
    logStatus("当前章节不是视频（可能是作业/讨论），跳到下一个未完成章节。");
    gotoNextUnfinished(index);
    return;
}
```

### 2. 添加防御性检查（可选但推荐）

在重试逻辑中，添加最大重试次数限制，防止无限循环：

* 添加一个重试计数器 `videoSwitchRetryCount`

* 如果重试超过3次仍然找不到视频，则按原逻辑跳转到下一章节

## 注意事项

* 切换后使用 `setTimeout` 延迟1.5秒等待界面切换完成

* 切换成功后递归调用 `next()` 重新开始检测流程

* 如果没有找到切换按钮或切换后仍找不到视频，则保持原有逻辑
